Цикл программ (скетчей) для Ардуино, в которых последовательно, начиная с самых простых конфигураций, иллюстрируется 
работа с индикаторами (светодиодами, лампочками, матрицами и др.). Цикл начинается с управляемого зажигания и гашения
встроенного светодиода Ардуино, далее рассматриваются дискретные внешние светодиоды, одно- и многоразрядные светодиодные 
индикаторы, матрицы, системы с несколькими индикаторами и многое другое. Используется плата Arduino UNO, подключённая 
к компьютеру через USB. Команды для управления индикаторами вводятся в командной строке, роль которой играет монитор 
последовательного порта в Arduino IDE.
